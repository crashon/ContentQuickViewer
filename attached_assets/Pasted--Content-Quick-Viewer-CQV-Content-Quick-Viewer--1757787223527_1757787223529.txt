# Content Quick Viewer (CQV)

**빠르고 직관적인 멀티미디어 컨텐츠 뷰어**

Content Quick Viewer는 폴더 내 다양한 컨텐츠(텍스트, 이미지, 동영상, 음악)를 빠르게 탐색하고 확인할 수 있는 경량 데스크톱 애플리케이션입니다.

## 🚀 주요 기능

### 📂 파일 탐색
- **폴더 트리 구조**: 직관적인 좌측 폴더 탐색
- **드래그&드롭**: 폴더를 쉽게 추가
- **최근 사용 폴더**: 자동으로 최근 폴더 목록 저장
- **빠른 검색**: 파일명 기반 실시간 검색

### 📝 텍스트 뷰어
- **다양한 포맷 지원**: TXT, MD, JSON, CSV, LOG, 코드 파일 등
- **Monaco Editor**: VS Code와 동일한 에디터 엔진
- **문법 강조**: 30+ 프로그래밍 언어 지원
- **빠른 검색**: 텍스트 내 문자열 검색 (Ctrl+F)
- **커스터마이징**: 폰트 크기, 자동 줄바꿈, 테마 설정

### 🖼️ 이미지 뷰어
- **포맷 지원**: JPG, PNG, GIF, WebP, SVG, TIFF 등
- **인터랙티브 조작**: 마우스 휠 확대/축소, 드래그 이동
- **슬라이드쇼**: 자동 이미지 전환 (Space 키)
- **회전 기능**: 90도 단위 회전
- **화면 맞춤**: 자동 크기 조절

### 🎬 동영상 뷰어
- **포맷 지원**: MP4, AVI, MKV, MOV, WebM 등
- **퀴 스크럽**: 진행률 바 클릭으로 빠른 이동
- **재생 제어**: 재생/일시정지, 볼륨 조절, 재생 속도 변경
- **단축키**: 5초/10초 점프, 전체화면 모드
- **자동 컨트롤**: 마우스 움직임에 따른 컨트롤 표시/숨김

### 🎵 음악 뷰어
- **포맷 지원**: MP3, WAV, FLAC, AAC, OGG 등
- **메타데이터**: 제목, 아티스트, 앨범 정보 표시
- **오디오 비주얼라이저**: 실시간 주파수 분석 시각화
- **재생 모드**: 반복 재생, 자동 다음 곡 재생
- **앨범 아트**: 기본 앨범 아트 표시

### ✏️ 파일 관리
- **파일명 수정**: F2 키 또는 더블클릭으로 인라인 편집
- **정렬 옵션**: 이름, 크기, 수정일, 타입별 정렬
- **보기 모드**: 격자 보기 / 목록 보기 전환
- **파일 정보**: 실시간 파일 정보 표시


### 2단계 (고급 기능)
- [ ] 다중 파일 일괄 이름 변경
- [ ] 썸네일 캐싱 시스템
- [ ] 다국어 지원 (한국어/영어/일본어)
- [ ] 라이트/다크 테마 전환
- [ ] 파일 즐겨찾기 기능

### 3단계 (확장 기능)
- [ ] 클라우드 스토리지 연동 (Google Drive, Dropbox)
- [ ] 태그 기반 파일 관리
- [ ] 전문 검색 (내용 검색, 메타데이터 검색)
- [ ] 플러그인 시스템
- [ ] 배치 작업 도구

## ⌨️ 단축키

| 기능 | 단축키 | 설명 |
|------|--------|------|
| 폴더 열기 | `Ctrl+O` | 폴더 선택 대화상자 열기 |
| 이전 파일 | `←` | 이전 파일로 이동 |
| 다음 파일 | `→` | 다음 파일로 이동 |
| 재생/일시정지 | `Space` | 미디어 재생 제어 |
| 파일명 변경 | `F2` | 선택된 파일 이름 편집 |
| 전체화면 | `F` | 동영상 전체화면 모드 |
| 음소거 | `M` | 오디오/비디오 음소거 |
| 검색 | `Ctrl+F` | 텍스트 파일 내 검색 |

### 미디어 전용 단축키
- `Shift+←/→`: 10초 점프
- `↑/↓`: 볼륨 조절
- `R`: 이미지 회전 / 음악 반복
- `0-4`: 동영상 재생 속도 변경


## 🔧 지원 파일 형식

### 텍스트 파일
- **일반**: TXT, MD, LOG, CSV, JSON, XML
- **코드**: JS, TS, PY, JAVA, C, CPP, PHP, RB, GO, RS 등
- **웹**: HTML, CSS, SCSS, LESS
- **설정**: INI, CFG, CONF, YAML, TOML

### 이미지 파일
- **래스터**: JPG, JPEG, PNG, GIF, BMP, TIFF, WebP
- **벡터**: SVG
- **아이콘**: ICO

### 동영상 파일
- **일반**: MP4, AVI, MKV, MOV, WMV
- **웹**: WebM, OGV
- **모바일**: 3GP, M4V

### 음악 파일
- **무손실**: FLAC, WAV
- **압축**: MP3, AAC, OGG, WMA, M4A, OPUS

- **다크 테마**: 눈의 피로를 줄이는 어두운 인터페이스
- **반응형 레이아웃**: 창 크기에 따른 자동 조절
- **분할 패널**: 드래그로 패널 크기 조절 가능
- **직관적 아이콘**: 파일 타입별 구분되는 아이콘
- **실시간 피드백**: 로딩, 에러 상태 표시
- **키보드 친화적**: 마우스 없이도 완전한 조작 가능
